# 赛博朋克游戏化量化系统 v5.0 设计文档

**日期：** 2026-05-08
**项目：** 个人量化交易系统
**设计风格：** 赛博朋克 + 游戏化体验

---

## 1. 设计概述

### 1.1 核心理念
将专业的量化交易系统包装成一个"量化任务控制台"，让用户感觉像是在操作飞船、执行战斗任务，同时保持核心功能的专业性和准确性。

### 1.2 视觉风格
- **主背景：** 深空黑 (#050508) + 网格纹理 + 数字雨效果
- **主色调：** 霓虹青 (#00f5ff) + 霓虹粉 (#ff00aa) + 警告橙 (#ff6600)
- **辅助色：** 数据绿 (#00ff88) + 危险红 (#ff3333)
- **特殊效果：** 扫描线、全息投影、霓虹光晕、故障艺术

### 1.3 交互特点
- 按钮点击有强烈的霓虹反馈
- 页面切换像切换飞船舱室
- 进度条像能量条，有充流动画
- 关键数据有全息投影效果

## 2. 页面设计详解

### 2.1 仪表盘 - 飞船主控制台
**游戏化概念：** 飞船驾驶舱，展示任务状态

**元素设计：**
- 顶部：当前策略名称 = 任务名称，带霓虹边框
- 左侧：账户余额 = HP条（数据绿渐变），最大回撤 = 护盾值（霓虹青）
- 中间：实时净值曲线 = 雷达扫描图，带扫描线动画
- 右侧：当前装备因子列表，每个因子带发光边框
- 底部：快速启动回测按钮 = "启动战斗"大按钮

### 2.2 策略配置 - 装备工坊
**游戏化概念：** 武器装备台

**元素设计：**
- 策略选择卡片 = 武器展示，hover时发出对应颜色的光
- 策略类型图标：
  - 动量策略 = 🔫 光束枪
  - 均值回归 = 🛡️ 护盾装置
  - 配对交易 = ⚔️ 双刃剑
- 参数滑块 = 能量调节杆，有刻度和数值显示
- 保存按钮 = "锻造装备"，点击有锻造动画

### 2.3 因子管理 - 弹药库
**游戏化概念：** 选择和装填弹药

**元素设计：**
- 因子卡片有稀有度边框：
  - 普通：白色边框
  - 稀有：霓虹青边框
  - 史诗：霓虹粉边框
  - 传说：金色边框
- 选择因子 = 装填弹药，有插槽插入动画
- IC/IR值 = 武器属性值，显示在卡片上
- 因子分类 = 弹药类型（技术/基本面/情绪等）

### 2.4 组合优化 - 战术分析
**游戏化概念：** 战场排兵布阵

**元素设计：**
- 优化方法选择 = 战术类型选择
- 权重表格 = 战场布阵，每个股票像一个士兵
- 优化过程 = 战术模拟，有计算动画和进度提示
- 优化结果 = "最佳战术方案"展示，带闪光效果
- 预期收益/波动率/夏普比率像战术评价指标

### 2.5 风险控制 - 护盾设置
**游戏化概念：** 防御系统配置

**元素设计：**
- 止损止盈 = 护盾能量等级，滑块像能量调节器
- 仓位限制 = 携带重量上限，进度条像载重指示
- 开关控件 = 护盾激活按钮，开时有充电动画
- 最大回撤限制 = 护盾容量，有刻度指示

### 2.6 回测执行 - 战斗模式
**游戏化概念：** 副本战斗

**元素设计：**
- 进度条 = 战斗HP，回测进度像战斗进程
- 实时收益 = 伤害输出数字，红色绿色跳动
- 关键事件提示：
  - "暴击！" = 单日大额收益
  - "躲避成功" = 成功止损
  - "连击！" = 连续盈利
- 完成后有"战斗胜利"动画和评分
- 可以暂停/继续 = 暂停战斗/恢复战斗

### 2.7 结果分析 - 战利品结算
**游戏化概念：** 战斗结束后的结算界面

**元素设计：**
- 绩效指标 = 战斗评分（S/A/B/C级）
  - S级：金色发光
  - A级：霓虹粉发光
  - B级：霓虹青发光
  - C级：普通显示
- 回撤分析 = 受伤记录，标注受伤严重程度
- 交易明细 = 击杀列表，显示每笔交易的"击杀"收益
- "解锁成就"提示区域：
  - 🏆 首次100%+收益
  - ⚡ 最大连胜记录
  - 🛡️ 完美风控（最大回撤<10%）

### 2.8 报告导出 - 任务日志归档
**游戏化概念：** 存档和日志管理

**元素设计：**
- 模板选择 = 日志格式选择
  - 完整报告 = 📜 完整任务日志
  - 绩效摘要 = 📊 战斗统计
  - 因子分析 = 🔬 装备分析
  - 交易日志 = 📝 击杀记录
- 导出选项 = 存档内容勾选
- 导出按钮 = "存档"按钮，点击有文件归档动画
- 实时预览 = 日志预览区域，带打字机效果

@keyframes matrixRain {
    0% { transform: translateY(-100%); opacity: 1; }
